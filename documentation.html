Guia rápido da API — SimpleEcommerce

MVP para autenticação, criação de loja e gestão de páginas (header, footer, conteúdo e rotas).

Base URL

http://localhost:3000

Troque para a URL do seu ambiente quando estiver em produção.

Autenticação (Cookie)

As rotas protegidas esperam um cookie de sessão definido por /login ou /register com keepMeLoggedIn=true.

Bearer One-Time

Algumas rotas exigem header Authorization: Bearer <token>, obtido via /token.

Auth

POST /register

Registra usuário e (se keepMeLoggedIn=true) já define o cookie de sessão.

POST /register
Content-Type: application/json

{
  "email": "ana@example.com",
  "firstName": "Ana",
  "lastName": "Silva",
  "password": "SenhaF0rte!",
  "keepMeLoggedIn": true
}
POST /login

Autentica e grava cookie de sessão quando keepMeLoggedIn=true.

POST /login
Content-Type: application/json

{ "email":"ana@example.com", "password":"SenhaF0rte!", "keepMeLoggedIn": true }
GET /me

Retorna o usuário autenticado (requer cookie).

GET /me
GET /token

Emite um token One-Time (requer cookie). Use no header Authorization.

GET /token
Authorization: (cookie já enviado pelo navegador)

Lojas

POST /stores
POST /stores
Authorization: Bearer <ONE_TIME_TOKEN>
Content-Type: application/json

{
  "name": "Minha Loja",
  "handle": "minha-loja",
  "primary_domain": "www.minhaloja.com"
}
GET /stores/:storeId
GET /stores/1
Authorization: Bearer <ONE_TIME_TOKEN>

Páginas

POST /stores/:storeId/pages
POST /stores/1/pages
Authorization: Bearer <ONE_TIME_TOKEN>
Content-Type: application/json

{
  "path": "/blog/post-1",
  "name": "Post 1",
  "type": "static",
  "header": { "menu":[{"label":"Início","href":"/"}] },
  "footer": { "text":"© 2025 Minha Loja" },
  "content": { "title":"Primeiro post" },
  "seo": { "title":"Post 1 • Minha Loja" },
  "status": "published"
}
GET /stores/:storeId/pages
GET /stores/1/pages?status=published
Authorization: Bearer <ONE_TIME_TOKEN>
GET /stores/:storeId/pages/... (middleware)
# /sobre
GET /stores/1/pages/sobre
Authorization: Bearer <ONE_TIME_TOKEN>

# /blog/post-1
GET /stores/1/pages/blog/post-1
Authorization: Bearer <ONE_TIME_TOKEN>
GET /stores/:storeId/page?path=...
GET /stores/1/page?path=%2F
Authorization: Bearer <ONE_TIME_TOKEN>

Códigos de status

  • 200 OK
  • 201 Created
  • 400 Requisição inválida
  • 401 Não autenticado
  • 403 Sem permissão
  • 404 Não encontrado
  • 409 Conflito
  • 500 Erro interno
